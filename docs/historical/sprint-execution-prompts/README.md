# Pre-Reset Sprint Execution Prompts

These files are preserved copies of the loose top-level `Ceal/prompts/` artifacts rescued on April 16, 2026.

They are historical prompt artifacts from the pre-reset sprint numbering regime. They are not the canonical active sprint prompts for current work. Current tracked sprint prompts live in `docs/sprints/`.

Why this folder exists:

- The top-level prompt files use the original sprint numbering and naming convention (`CLAUDE_CODE_*.md`).
- Several of them do not map 1:1 to the post-reset `docs/sprints/` files.
- Deleting the top-level tree without rescuing these files would lose portfolio-relevant planning artifacts that document how Sprints 1-10 were originally executed.

How to use these files:

- Treat them as historical references and interview/portfolio artifacts.
- Do not paste them into current implementation sessions without first reconciling scope, numbering, and current repo state.
- Prefer `docs/sprints/` for canonical in-repo execution prompts tied to the recovered post-reset codebase.
